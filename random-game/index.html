About game
This is not a classic Miner game. To win, you have to place all the flags on the mined tiles. When you set the last flag, if any of the flags are set wrong, you'll lose.
Also, you have a time limit. It's different on every level and board size.
Score tables are different for every level and board size too.
If you wish to win quickly, just click the "Let me win" checkbox in the menu
Good Luck!
